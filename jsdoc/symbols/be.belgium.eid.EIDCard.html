Class Detail
be.belgium.eid.EIDCard()
EIDCard contains the public readable identity data of an eID card. Almost all the properties of this object are of type String, Number, Date or Boolean. Some properties (sex, documentType, specialStatus) have types which are especially defined in this library.

The applet from eID middleware 2.5.9 / 2.6 provided the methods getFirstName1, getFirstName2 and getFirstName3. The applet from eID middleware 3.5 has only one method, getFirstName, which returns the first, second and third first names. The third first name on the eID card is only the first letter of the third name. Some people have a first name consisting of two words. For example Pieter Jan or Jean Marie. So it is impossible to determine the second first name. Still the EIDCard object has the firstName2 property in order to be backwards compatible with the applet from eID middleware 2.5.9 / 2.6. But the firstName2 property is always an empty string. Maybe in a future middleware, the applet will return the first names correctly.

Formats of identity data on an eID card are described in the following documents:

See:
belgian_electronic_identity_card_content_v2.8.a.pdf
formaat_van_de_datums_04042006.pdf
