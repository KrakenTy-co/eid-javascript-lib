When an eID card is read, the hidden input field will contain the base64 url safe encoded picture.
The base64 encoding only works in Internet Explorer, if JRE 6 Update 10 (Java Runtime Environment 1.6.0_10) or higher is installed
and if the Next-Generation Java Plug-in is enabled (this is the default setting) in the Java Control Panel.

The code for base64.js can be downloaded at http://hellerim.net/base64_src.php
Small modifications needs to be done. But please, keep the copyright statement intact.
Put the following code in comments:
/*
try { 
  if (system.utility.base64 == null) {
    throw new Error('');
  };
} catch(e) { // don't install twice
  // dependencies
  try {
    eval('core');
  } catch(e) {
    throw new Error('base64: class \'core\' not installed');
  }
*/
and at the bottom line:
/*
core.installClass(base64);

} // end catch
*/
