Class Detail
be.belgium.eid.SISCard()
SISCard contains the public readable identity data of a SIS card. Almost all the properties of this object are of type String, Number, Date or Boolean. Except property sex that has a type which is especially defined in this library.

Formats of identity data on an SIS card are described in the following documents:

See:
http://www.ksz-bcss.fgov.be/nl/documentation/document_3.htm#document3_3
http://www.ksz-bcss.fgov.be/documentation/nl/documentation/appareils%20de%20lecture%20carte%20SIS/td-002bis-nl.pdf
